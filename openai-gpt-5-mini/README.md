# Sprint 1 MVP - Mensaplan (Scaffold)

Dieses Verzeichnis enthält ein minimales Flutter-Projekt-Scaffold zur Umsetzung von Sprint 1 (Mensaplan). Es enthält:

- Projektstruktur (Features, Shared, Models)
- Basis-API-Client für OpenMensa (Canteen ID 6)
- Riverpod Provider-Beispiel
- Basis-Widgets: Loading / Error / Empty
- Tab-Navigation mit Placeholder-Screens

Anmerkung: Dies ist ein Code-Scaffold; zum Ausführen wird Flutter SDK benötigt (`flutter run`).

Quick start:

```bash
cd livedemo-template/openai-gpt-5-mini
flutter pub get
flutter run
```
